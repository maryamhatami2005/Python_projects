# Monty Hall simulation
This project provides a simulation of the Monty Hall problem, a classic probability puzzle based on a game show scenario.
The simulation lets you compare your chances of winning the car by switching vs not switching doors after the host reveals a goat.

## Requirements
- python 3.7+
- streamlit(optional)
  
## Project Structure

```
├── README.md
├── images
└── src
    ├── app.py
    ├── monty_hall.py
    └── test.ipynb
```

## Usage
You can play the Monty hall game simulation by adding the `src` directory to the the PYTHONPATH and running:

` python src/monty_hall.py `

To start Streamlit dashboard, run:

` streamlit run src/app.py`
