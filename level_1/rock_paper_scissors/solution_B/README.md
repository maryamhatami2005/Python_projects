# Rock Paper Scissors (Functional Programming approach)
This project is a simple implementation of the classic **Rock–Paper–Scissors** game, built using functional programming principles.

## Project structure
```
rock_paper_scissors
├── README.md
└── src
    ├── main.py
    └── test.ipynb
```

## Requirements
- Python 3.7+

## Running the Project
To run the project:

```python
python/src/main.py
```
